# Семестровая работа по теме TIMSORT

В проекте есть:
- код алгоритма TIMSORT в упрощённой учебной версии;
- генератор 100 входных файлов;
- программа для измерения времени и количества шагов;
- готовые результаты и графики;
- отчёт.

## Как запустить

Сначала установите библиотеку matplotlib:

```bash
pip install matplotlib
```

Потом по очереди запустите:

```bash
python src/data_generator.py
python src/benchmark.py
```

## Что делает каждая программа

- `src/data_generator.py` — создаёт 100 файлов со случайными массивами.
- `src/timsort.py` — сортирует массив алгоритмом TIMSORT.
- `src/benchmark.py` — читает файлы, запускает сортировку, считает время и шаги, строит графики.
